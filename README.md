# SQL Data Analysis Project – Sales Database

This project demonstrates how SQL is used to analyze business sales data.
The goal is to find revenue trends, customer behavior, and product performance.

## Dataset
The dataset contains:
- Order ID  
- Customer Name  
- Product  
- Category  
- Sales  
- Quantity  
- Profit  
- Order Date  
- Region  

## Objectives
- Find total sales and profit  
- Identify top customers  
- Analyze product & category performance  
- Monthly sales trends  
- Regional performance  


## Tools Used
- Microsoft SQL Server (SSMS)
- SQL (Joins, Group By, Subqueries, Aggregations)


## Files
- dataset/sales_data.csv → Raw data  
- queries/sql_queries.sql → All SQL queries used for analysis  


## Sample Insights
- Top 5 customers by revenue  
- Best-selling products  
- Monthly sales growth  
- Highest profit region
